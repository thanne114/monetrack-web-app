# Monetrack Web App - Buku Kas Digital

Aplikasi berbasis web sederhana untuk mencatat pemasukan dan pengeluaran harian guna melacak *cashflow* secara digital. Proyek ini dibuat untuk memenuhi tugas mata kuliah di Semester 4, program studi Teknik Informatika.

## 🚀 Fitur Utama
*   **Full CRUD**: Tambah, Tampil, Edit, dan Hapus transaksi keuangan.
*   **Visualisasi Data**: Diagram lingkaran (*Doughnut Chart*) menggunakan Chart.js yang menunjukkan rasio pemasukan vs pengeluaran.
*   **Filter Periode**: Menyaring riwayat transaksi dan diagram berdasarkan rentang tanggal tertentu.
*   **Responsive Design**: Antarmuka modern menggunakan Bootstrap 5.

## 🛠️ Teknologi yang Digunakan
*   **Bahasa Pemrograman**: PHP (Native)
*   **Database**: MySQL
*   **Styling**: Bootstrap 5 & FontAwesome
*   **Library Chart**: Chart.js

## ⚙️ Cara Instalasi
1. Clone repository ini.
2. Impor file database `db_keuangan.sql` melalui phpMyAdmin.
3. Sesuaikan konfigurasi database di file `koneksi.php`.
4. Jalankan aplikasi melalui XAMPP atau web server lokal lainnya.
